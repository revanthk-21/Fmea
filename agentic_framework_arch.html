S
Supervisor Node — Entry Point
User request hits FastAPI → packaged as AgentState TypedDict with task_type field. LangGraph supervisor reads task_type and routes via conditional edge to one of three sub-graphs. Checkpointer (Redis/SQLite) saves state for resume on failure.
R
RAG Agent Flow
embed_query → retrieve_docs → grade_relevance → generate → return Query is embedded via Titan. FAISS MMR retrieval fetches top-k diverse chunks. Optional relevance grading node filters low-quality chunks. Claude Sonnet generates answer with retrieved context. Supports multi-hop: if answer confidence low, reformulates query and retrieves again (cycle in LangGraph).
D
DFMEA Agent Flow
route_case → [new | new_env | design_change] → wizard_steps → export Case router reads dfmea_case from state. Each case triggers a different sub-graph path — your existing FastAPI routers are wrapped as LangGraph tool nodes. The wizard progresses through 10 steps (Elements→...→Review). LLM fills in failure modes, causes, and risk ratings at each step. Outputs structured XLSX via your existing exporter.
O
Optimization Agent Flow (Cyclic)
init_k → solve_ode → compute_rms → check_convergence → [loop | done] Starts with initial spring stiffness k. ODE node integrates quarter-car equations via solve_ivp (RK45). RMS cabin acceleration computed from output. Optuna Bayesian optimizer proposes new k. LangGraph conditional edge: if |ΔRMS| > ε and iterations < max, loop back to solve_ode. On convergence, LLM summarizes optimal k and RMS with engineering interpretation. Full iteration history streamed to frontend.
